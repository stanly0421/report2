# 快速使用指南

## 開始播放音樂

1. **載入音樂檔案**
   - 點擊頂部的「📁 載入音樂檔案」按鈕
   - 選擇 MP3, WAV, FLAC, M4A, OGG 或 AAC 格式的音樂檔案
   - 音樂將自動加入當前播放清單並開始播放

2. **播放控制**
   - **播放/暫停** (▶/⏸): 控制音樂播放
   - **上一首** (⏮): 播放前一首歌曲
   - **下一首** (⏭): 播放下一首歌曲
   - **隨機播放** (🔀): 啟用/停用隨機播放模式
   - **循環播放** (🔁): 啟用/停用循環播放模式

3. **進度控制**
   - 拖動進度條可跳轉到指定播放位置
   - 進度條下方顯示當前時間 / 總時長（mm:ss 格式）
   - 點擊字幕中的時間戳也可以跳轉

4. **音量控制**
   - 使用右下角的音量滑桿調整音量（0-100%）
   - 音量圖標會根據音量大小變化：
     - 🔇 靜音（0%）
     - 🔈 低音量（1-32%）
     - 🔉 中音量（33-65%）
     - 🔊 高音量（66-100%）

## 管理播放清單

1. **創建新播放清單**
   - 點擊左側面板的「➕ 新增」按鈕
   - 輸入播放清單名稱
   - 新播放清單會立即顯示在下拉選單中

2. **切換播放清單**
   - 使用左側面板頂部的下拉選單切換播放清單
   - 每個播放清單獨立管理歌曲

3. **刪除播放清單**
   - 點擊左側面板的「🗑️ 刪除」按鈕
   - 確認刪除（至少保留一個播放清單）

4. **管理歌曲**
   - **播放歌曲**: 雙擊播放清單中的項目
   - **調整順序**: 拖放歌曲項目重新排序
   - **移除歌曲**: 右鍵點擊項目，選擇「從播放清單移除」

## 使用我的最愛功能

1. **加入最愛**
   - 播放任何歌曲時，點擊「❤️ 加入最愛」按鈕
   - 歌曲會自動加入到「我的最愛」播放清單

2. **移除最愛**
   - 當按鈕顯示「💔 移除最愛」時，點擊可從最愛中移除
   - 在「我的最愛」播放清單中，按鈕始終顯示「💔 移除最愛」

3. **智能按鈕**
   - 按鈕會根據當前播放清單和歌曲狀態自動切換
   - 在「我的最愛」播放清單中: 顯示「💔 移除最愛」
   - 在其他播放清單中: 根據歌曲是否在最愛中顯示對應文字

## 字幕功能（需要 Whisper）

1. **自動轉錄**
   - 播放本地音樂時，Whisper 會自動開始轉錄
   - 字幕會即時顯示在音樂資訊下方

2. **跳轉功能**
   - 點擊字幕中的時間戳（如 [0.00s - 5.23s]）
   - 播放器會立即跳轉到該時間點
   - 配合進度條使用可精確控制播放位置

## 鍵盤快捷鍵

目前不支援鍵盤快捷鍵，所有操作通過滑鼠點擊完成。

## 提示與技巧

- **精確跳轉**: 使用字幕時間戳進行精確跳轉
- **快速調整音量**: 直接拖動音量滑桿，無需點擊
- **組織歌曲**: 使用多個播放清單分類管理不同類型的音樂
- **快速播放**: 雙擊播放清單中的歌曲立即播放
- **清單排序**: 拖放歌曲可以自由調整播放順序

## 常見問題

**Q: 為什麼我的播放清單沒有編號？**
A: 這是最新版本的設計改進，移除編號讓介面更簡潔。

**Q: 如何知道歌曲是否在最愛中？**
A: 播放歌曲時，查看右下角的按鈕。如果顯示「💔 移除最愛」，表示已在最愛中。

**Q: 進度條不能拖動怎麼辦？**
A: 確保有音樂正在播放。進度條只在有音樂播放時才能使用。

**Q: 音量調整沒有反應？**
A: 檢查系統音量設置。應用內音量控制與系統音量設置相互獨立。

**Q: 字幕沒有顯示？**
A: 確保已安裝 Python 3 和 openai-whisper 模組。字幕功能是可選的。
