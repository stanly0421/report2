## 顏色方案 (Spotify 風格)

### 主要顏色
- **背景色**: `#121212` (深黑色)
- **次要背景**: `#181818` (稍淺黑色)
- **面板背景**: `#000000` (純黑色)
- **控制元件背景**: `#282828` (灰黑色)

### 文字顏色
- **主要文字**: `#FFFFFF` (純白色)
- **次要文字**: `#B3B3B3` (淺灰色)
- **邊框**: `#404040` (深灰色)

### 強調色
- **主要強調**: `#1DB954` (Spotify 綠)
- **懸停效果**: `#1ED760` (淺綠色)
- **按下效果**: `#1AA34A` (深綠色)
- **選中背景**: `#1DB954` (綠色背景)

```css

背景: #282828 (灰色)
圖示: 14px, 白色
```

```css

已啟用:
  - 背景: #1DB954 (綠色)
  - 圖示: 白色
  - 表示功能已開啟
```

## 互動效果

### 1. 懸停效果 (Hover)
- 背景色變淺或變亮
- 文字顏色從灰色變白色
- 過渡動畫: 0.2s ease

### 2. 點擊效果 (Active)
- 背景色變深
- 輕微的按下感覺

### 3. 選中效果 (Selected)
- 綠色背景 (#1DB954)
- 白色文字
- 粗體字型（播放中的項目）

### 4. 停用效果 (Disabled)
- 背景: #181818 (深灰)
- 文字: #404040 (暗灰)
- 無法點擊

## 動畫效果

### 過渡動畫
```css
transition: all 0.2s ease;
```
應用於:
- 按鈕懸停
- 背景色變化
- 文字顏色變化

### 無動畫的元素
- 影片播放器
- 搜尋結果載入
- 列表項目添加

## 可訪問性 (Accessibility)

### 對比度
- 文字與背景對比度 > 4.5:1 (WCAG AA)
- 綠色按鈕與白色文字對比度 > 4.5:1

### 鍵盤導航
- Tab 鍵可遍歷所有互動元素
- Enter 鍵可執行主要操作
- 箭頭鍵可在列表中導航

### 工具提示
- 圖示按鈕顯示工具提示
- 說明按鈕功能

## UI 優勢

### 1. 美觀性
- 採用流行的 Spotify 設計風格
- 深色主題減少眼睛疲勞
- 綠色強調色醒目但不刺眼

### 2. 易用性
- 佈局清晰，功能分區明確
- 常用功能觸手可及
- 視覺層次分明

### 3. 一致性
- 統一的配色方案
- 一致的按鈕樣式
- 規範的間距系統

### 4. 響應式
- 支援視窗大小調整
- 面板可自由調整比例
- 內容自適應

## 設計靈感來源

- **Spotify Desktop App**: 整體佈局和配色
- **YouTube Music**: 搜尋和播放清單功能
- **Material Design**: 陰影和互動效果
- **Fluent Design**: 圓角和透明度概念

這個 UI 設計結合了現代音樂串流平台的最佳實踐，提供了美觀、直覺且功能豐富的使用體驗。
